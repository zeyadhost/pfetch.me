# pfetch.me

A customizable bio-link platform built with Next.js and Firebase.

## Stack

- **Next.js**
- **Tailwind CSS v4**
- **Firebase**
- **Cloudflare Pages**

## Dev Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Project Structure

```
src/
└── app/
    ├── page.tsx
    ├── layout.tsx
    └── globals.css
```
